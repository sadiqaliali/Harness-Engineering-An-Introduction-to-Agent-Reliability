# Contributing to Harness Kernel

We are building the infrastructure layer for reliable AI agents. Since this is an active research project, we value contributions that:
1. Improve reliability benchmarks.
2. Enhance the cryptographic integrity of the Flight Recorder.
3. Propose new, verifiable consensus models for the Jury.

To contribute:
1. Fork the repo.
2. Create a feature branch.
3. Submit a Pull Request.

Please ensure all contributions include updated unit tests in the `/tests` folder.
