# Harness Engineering: Comprehensive MCQ Bank

## Chapter 1: Introduction to Harness Engineering
1. **What is the primary goal of Harness Engineering?**
   - A) To increase the speed of LLM inference.
   - B) To build operational infrastructure for reliable and governable AI agents.
   - C) To replace human programmers with autonomous agents.
   - D) To create better prompt templates.
   - *Answer: B*

2. **In the automobile analogy, "Prompt Engineering" is the engine. What is "Harness Engineering"?**
   - A) The fuel.
   - B) The driver.
   - C) The chassis, brakes, and safety infrastructure.
   - D) The destination.
   - *Answer: C*

## Chapter 3: Agent Reliability Engineering (ARE)
3. **What are the four pillars of ARE?**
   - A) Prompting, Chaining, RAG, and Fine-tuning.
   - B) Determinism, Auditability, HITL, and Fail-Safe Mechanisms.
   - C) Speed, Cost, Accuracy, and Scale.
   - D) Python, Docker, Kubernetes, and Terraform.
   - *Answer: B*

4. **The "Confidence Chain" refers to:**
   - A) The sequence of prompts sent to a model.
   - B) The requirement that every reasoning step has a quantified confidence metric.
   - C) The speed of the internet connection.
   - D) The number of agents in a team.
   - *Answer: B*

## Chapter 5: The Agent Flight Recorder (AFR)
5. **How does the AFR ensure tamper-evidence?**
   - A) By encrypting the entire log with a password.
   - B) By using cryptographic hash-links (SHA-256) between entries.
   - C) By saving the logs on a read-only CD-ROM.
   - D) By hiding the log file in a secret directory.
   - *Answer: B*

6. **What does the AFR capture that standard logs often miss?**
   - A) The timestamp.
   - B) The reasoning graph and intermediate intent.
   - C) The CPU usage.
   - D) The user's IP address.
   - *Answer: B*

## Chapter 6: Governance & Policy
7. **What is "Policy-as-Code" (PaC) in the context of Harness?**
   - A) Writing legal contracts in Python.
   - B) Enforcing safety boundaries via declarative rules injected at the runtime kernel.
   - C) Asking the agent to "be nice" in the system prompt.
   - D) Using AI to write the company's privacy policy.
   - *Answer: B*

8. **A "Circuit Breaker" in the Harness Runtime triggers when:**
   - A) The computer runs out of battery.
   - B) The agent's confidence or safety score falls below a set threshold.
   - C) The LLM API returns a 500 error.
   - D) The user closes the terminal.
   - *Answer: B*
